# Tableau Dashboard Creation Instructions

Since Tableau workbooks (.twbx) must be created using the Tableau Desktop or Public software, please follow these steps to build the requested panels:

### Data Source
Connect Tableau to the `data/ev_dataset.csv` file generated by `generate_data.py`.

### Required Visualizations

1. **Charging Heatmap (Locations vs Time)**
   - **Columns:** `charging_time_hour` (Continuous Dimension)
   - **Rows:** `location`
   - **Color:** `Count(ev_dataset.csv)` or `SUM(power_consumed_kwh)`
   - **Type:** Density/Heatmap

2. **Charging Time Histogram**
   - **Columns:** `charging_time_hour` 
   - **Rows:** `Count`
   - **Type:** Bar Chart

3. **Battery Performance vs Range Scatter Plot**
   - **Columns:** `dod_pct` (Depth of Discharge)
   - **Rows:** `range_actual_km`
   - **Detail:** `vehicle_id`
   - **Type:** Scatter Plot (Add Trend Line)

4. **EV Model Efficiency Comparison Bar Chart**
   - **Columns:** `ev_model`
   - **Rows:** `AVG(efficiency_kwh_per_km)`
   - **Type:** Bar Chart

5. **Cost per Kilometer Chart**
   - **Columns:** `ev_model`
   - **Rows:** `AVG(cost_per_km_usd)`
   - **Type:** Bar Chart

6. **Predicted vs Actual Range**
   - **Columns:** `Measure Names`
   - **Rows:** `Measure Values` (`range_predicted_km`, `range_actual_km`)
   - **Color:** `Measure Names`
   - **Type:** Side-by-side Bar

### Create the Dashboard
1. Create a new Dashboard.
2. Drag and drop all the above sheets onto the dashboard.
3. Add a filter for `ev_model` and apply it to "All Using This Data Source".
4. Add a filter for `location`.

### Create the Story
1. Create a new Story.
2. Scene 1 (Charging Patterns): Drag in the generic charging use visualizations (Heatmap, Histogram).
3. Scene 2 (Battery Health): Use the Scatter Plot showing DoD vs Range.
4. Scene 3 (Model Comparison): Bring in the efficiency and cost comparisons.

### Embedding
Publish the resulting workbook to Tableau Public. Note the URL. Update the Flask template file `dashboard.html` and `story.html` to point to your new Published URL.
